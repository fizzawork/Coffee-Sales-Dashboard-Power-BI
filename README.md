# ☕ Coffee Sales Analysis Dashboard – Power BI

An interactive Power BI dashboard designed to analyze coffee shop sales, understand customer purchase behavior, and identify key revenue trends across different time periods, products, and payment methods.

## Project Overview

This project visualizes the Coffee Sales dataset using Power BI to uncover insights such as:

-Which coffee types generate the highest revenue

-How sales vary by hour of day, day of week, and month

-Customer preferences based on payment method

-Time-of-day analysis (Morning, Afternoon, Evening, Night)

-Sales trends and performance metrics over time

The dashboard is fully interactive, allowing users to explore sales patterns and gain a deeper understanding of operational performance.

## Key Features & Insights
1. Sales Performance Overview

-Total revenue, total orders, average revenue per order

-Highest and lowest order values

-Daily average sales trends

2. Coffee Products Breakdown

-Money earned by each coffee type

-Number of orders per coffee

-Best-selling vs. least-selling coffee beverages

3. Time-Based Analysis

-Hourly peak sales analysis

-Weekday performance with sorted day names

-Monthly revenue tracking with proper chronological sorting

4. Customer Payment Behavior

-Revenue split by payment method (Card, Cash, Wallet, etc.)

-Transaction trends segmented by payment type

5. Time-of-Day Insights

-Sales groups: Morning → Afternoon → Evening → Night

-Behavior analysis based on consumption patterns
